# Kanban Board 📋✨

Aplicação web desenvolvida utilizando apenas **HTML**, **CSS** e **JavaScript**.

🔗 Acesse: https://kanban.vlopinhos.dev/

## 📌 Sobre o Projeto

Este projeto consiste em um **Kanban simples**, porém com interações bem definidas e foco na organização visual de tarefas.

A aplicação permite organizar atividades em colunas, simulando um fluxo de trabalho tradicional (ex: A Fazer, Em Progresso, Concluído).

## 🚀 Funcionalidades

- ➕ Criação de tarefas  
- 🗂 Organização por colunas  
- 🔄 Movimentação de cards entre etapas  
- 🖱 Interações dinâmicas com JavaScript  
- 🎨 Interface limpa e intuitiva  

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

## 🎯 Objetivo

O principal objetivo foi praticar:

- Manipulação do DOM  
- Controle de eventos  
- Estruturação de layout responsivo  
- Organização de código em JavaScript puro  

Projeto focado em simplicidade, usabilidade e boa experiência visual.
